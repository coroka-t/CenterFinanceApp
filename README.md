# CenterFinanceApp

Данный проект представляет собой учебное Java-приложение, предназначенное для автоматизации финансового учета в детском центре (или ИП, работающем в сфере дополнительного образования). Основная идея проекта — заменить ручной учет (таблицы Excel или бумажные записи) на простое и удобное программное решение, которое позволяет отслеживать доходы, контролировать оплаты воспитанников и своевременно выявлять задолженности. Проект разработан в рамках учебной работы и демонстрирует применение объектно-ориентированного программирования, работу с коллекциями, файловым вводом/выводом и базовыми принципами учета данных.

**Целью проекта** является создание прототипа приложения, которое:
- помогает владельцу или администратору детского центра вести учет доходов;
- обеспечивает хранение информации о воспитанниках и внесенных платежах;
- автоматизирует процесс отслеживания задолженностей;
- закрепляет практические навыки программирования на языке Java.

Основной функционал
- **Управление воспитанниками**: добавление, просмотр, редактирование и удаление информации о детях (ФИО, возраст, контактные данные родителей, группа);
- **Учет платежей**: фиксация поступлений (сумма, дата, период оплаты, привязка к конкретному ребенку);
- **Отслеживание задолженностей**: автоматическое определение воспитанников, у которых отсутствует оплата за текущий период;
- **Просмотр статистики**: отображение общего дохода за выбранный период (день/неделя/месяц);
- **Хранение данных**: сохранение информации в файлах или базе данных для последующего использования между сессиями;
- **Простой интерфейс**: консольное меню для взаимодействия с пользователем.

Используемые технологии
- **Java 21** — язык программирования;
- **Java Collections Framework** — для хранения и обработки данных в оперативной памяти;
- **Git и GitHub** — для контроля версий.
