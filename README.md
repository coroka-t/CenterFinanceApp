# CenterFinanceApp 

 Описание проекта: 

**CenterFinanceApp** — это Java-приложение для автоматизации финансового учета в детских развивающих центрах и организациях дополнительного образования. Программа предназначена для владельцев малого бизнеса, администраторов и руководителей, которые хотят заменить ручной учет (таблицы Excel или бумажные записи) на простое и надежное программное решение.

Приложение позволяет вести базу воспитанников, регистрировать поступления оплат, отслеживать задолженности и формировать базовую финансовую статистику. Проект разработан в учебных целях и демонстрирует практическое применение объектно-ориентированного программирования, работу с коллекциями и файловым вводом/выводом на языке Java.

Основные возможности:

- **Управление воспитанниками**: добавление, просмотр, редактирование и удаление записей о детях (ФИО, возраст, контактные данные родителей).
- **Учет платежей**: фиксация поступлений с указанием суммы, даты, периода оплаты и привязкой к конкретному ребенку.
- **Отслеживание задолженностей**: автоматическое выявление воспитанников, не имеющих оплаты за текущий период.
- **Просмотр статистики**: отображение общего дохода за выбранный период (день, неделя, месяц).
- **Сохранение данных**: хранение информации между сессиями с помощью файловой системы (поддержка БД в разработке).

 Технические требования:

Для запуска и разработки приложения необходимо следующее программное обеспечение:

- **Java Development Kit (JDK)** — версия 21 или выше
- **Операционная система**: Windows 10/11, macOS, Linux (любая с поддержкой Java)
- **Сборщик**: Maven (рекомендуется) или Gradle
- **Среда разработки**: IntelliJ IDEA
- **Git** — для клонирования репозитория и работы с версиями
